Every formula follows AIA Document G702/G703 exactly: Total Completed, Retainage, Earned Less Retainage, Current Payment Due, Balance to Finish. The math is independently tested against 51 verification checks before every release.

### Every Pay App, Every Period — Tracked Automatically
Previous billing percentages carry forward automatically. The system knows what was billed last period and prevents double-counting. Your full billing history lives in one place.

### Print-Perfect PDFs

### Flexible Retainage
Set a default retainage rate per project — or override it per line item. Support for 0% retainage on lump-sum contracts, mixed-rate schedules, and anything in between.

### Change Orders Built In
Track approved change orders against each pay application. The contract sum updates automatically, and all G702 summary lines recalculate instantly.

## FAQ

**Does it handle multiple pay applications on the same project?**
Yes. Each pay period you create a new pay application. Previous billing percentages carry forward automatically — you only enter what's new this period.

**What file formats does it accept for the Schedule of Values?**
Excel (.xlsx, .xls), CSV, PDF contractor estimates, and Word documents (.docx). Our AI parser handles unstructured contractor templates — not just perfectly formatted spreadsheets.

**Is the math actually accurate? This is real money.**
Yes. Every formula is independently verified against 51 test cases before each release, covering standard billing, multi-period roll-forwards, zero retainage, stored materials, change orders, and edge cases. The formulas match AIA Document G702/G703 exactly.

**Can I set a retainage rate other than 10%?**
Yes. Set any retainage rate per project (including 0%), and override it per line item for mixed-rate schedules.

**Is my data secure?**
All data is stored in a private PostgreSQL database behind JWT authentication. Your projects and billing history are only accessible to your account.

**Do I need to install anything?**
No. It runs entirely in your web browser. Works on Mac, PC, and tablet.
